开启流量捕获

捕获队伍解题流量 需开启平台代理

平台代理配置 https://docs.ctf.gzti.me/zh/guide/platform-proxy

在 appsettings.json 中，找到 ContainerProvider 节点，进行如下配置：

{ “ContainerProvider”: { “PortMappingType”: “PlatformProxy”, “EnableTrafficCapture”: false } }

然后使用 websocket进行连接
